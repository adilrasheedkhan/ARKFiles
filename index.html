Sometimes you need to add pictures and captions to your site but you don’t want to entirely reformat the whole thing or get into the CSS (cascading style sheet). Often, the perfect solution is creating a table. Tables can work nearly anywhere you can enter HTML coding. Be sure you are using the HTML editing window, not the visual. You need the HTML window because it will read the coding you’ll use for your table. Every table starts with the word in brackets, followed by the letters which stand for ‘table row’. Then comes the
which stands for ‘table data cell’. Your information will be after this table data tag. These are all called ‘tags’ and they all must be closed where they finish by repeating the tag with a / inserted.
